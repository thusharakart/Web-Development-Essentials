Setup for windows

1. open cmd
2. > psql -U postgres 
3. enter passward
4. To view the list of databases 
    > \l
5. To create a database
    > CREATE DATABASE databasename;
6. To check list of roles
    > \du
7. To connect to the database
    > \c databasename
8. To check the table description
    > \d tablename
